# C++零基础学习记录
记录知识盲点

## 信息的储存单位
* 位（bit,b)
> 数据的最小单位，表示一位二进制信息
* 字节(byte,B)
> 八位二进制数字组成（1byte=8bit）

## R进制缩写
* 二进制：BIN
* 八进制：OCT
* 十进制：DEC
* 十六进制：HEX

## R进制->十进制
各位数字与它的权相乘，其积相加
> 例：二进制的1111转换成十进制  
1×2<sup>3</sup>+1×2<sup>2</sup>+1×2<sup>1</sup>+1×2<sup>0</sup>=8+4+2+1=15

## 十进制->R进制
除以R取余
> 例：十进制的68转换成二进制  
68/2=34(余数0）34/2=17(余数0）17?2=8(余数1）……1/2=0(余数1） 
低位->高位
得：1000100
